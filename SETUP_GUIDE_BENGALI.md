# হোটেল রুম ম্যানেজমেন্ট সিস্টেম - সেটআপ গাইড

## ধাপ ১: Node.js ইনস্টল করুন

1. [Node.js](https://nodejs.org/) ওয়েবসাইটে যান
2. LTS ভার্সন ডাউনলোড করুন
3. ইনস্টল করুন এবং কম্পিউটার রিস্টার্ট করুন

## ধাপ ২: MongoDB সেটআপ করুন (ফ্রি ক্লাউড ডেটাবেস)

1. [MongoDB Atlas](https://www.mongodb.com/cloud/atlas) এ যান
2. "Try Free" বাটনে ক্লিক করুন এবং একাউন্ট তৈরি করুন
3. একটি ফ্রি ক্লাস্টার তৈরি করুন:
   - Provider: AWS
   - Region: যেকোনো (Singapore বা Mumbai কাছাকাছি)
   - Cluster Tier: M0 Sandbox (FREE)
4. ডেটাবেস ইউজার তৈরি করুন:
   - Username: admin
   - Password: যেকোনো শক্তিশালী পাসওয়ার্ড (মনে রাখুন!)
5. Network Access সেটআপ:
   - "Add IP Address" ক্লিক করুন
   - "Allow Access from Anywhere" সিলেক্ট করুন
   - Confirm করুন
6. Connection String কপি করুন:
   - "Connect" বাটনে ক্লিক করুন
   - "Drivers" সিলেক্ট করুন
   - Connection string কপি করুন
   - `<password>` এর জায়গায় আপনার পাসওয়ার্ড দিন

## ধাপ ৩: জিপ ফাইল এক্সট্র্যাক্ট করুন

1. ডাউনলোড করা ZIP ফাইলে রাইট ক্লিক করুন
2. "Extract All" বা "Extract Here" সিলেক্ট করুন
3. এক্সট্র্যাক্ট করা ফোল্ডার খুলুন

## ধাপ ৪: ব্যাকএন্ড সেটআপ করুন

1. ফোল্ডারে `backend` ফোল্ডার খুলুন
2. এই ফোল্ডারে রাইট ক্লিক করুন এবং "Open in Terminal" বা "Git Bash Here" সিলেক্ট করুন
3. নিচের কমান্ডগুলো একটা একটা করে রান করুন:

```bash
npm install
```

4. `.env.example` ফাইল কপি করে `.env` নাম দিন
5. `.env` ফাইল খুলুন (Notepad বা যেকোনো টেক্সট এডিটর দিয়ে)
6. MongoDB connection string পেস্ট করুন:

```
MONGODB_URI=mongodb+srv://admin:your_password@cluster0.xxxxx.mongodb.net/hotel-management
PORT=5000
```

7. টার্মিনালে ব্যাকএন্ড সার্ভার স্টার্ট করুন:

```bash
npm run dev
```

8. যদি সফল হয়, দেখবেন: "Server running on port 5000" এবং "MongoDB Connected"

## ধাপ ৫: ফ্রন্টএন্ড সেটআপ করুন

1. একটি **নতুন টার্মিনাল/কমান্ড প্রম্পট** খুলুন
2. মূল ফোল্ডারে `frontend` ফোল্ডার খুলুন
3. এই ফোল্ডারে টার্মিনাল খুলুন
4. নিচের কমান্ডগুলো রান করুন:

```bash
npm install
```

5. `.env.local.example` কপি করে `.env.local` নাম দিন (ভিতরে ইতিমধ্যে সঠিক কনফিগারেশন আছে)
6. ফ্রন্টএন্ড সার্ভার স্টার্ট করুন:

```bash
npm run dev
```

7. দেখবেন: "Ready - started server on 0.0.0.0:3000"

## ধাপ ৬: অ্যাপ্লিকেশন ব্যবহার করুন

1. ব্রাউজার খুলুন (Chrome, Firefox, etc.)
2. এই অ্যাড্রেসে যান: `http://localhost:3000`
3. "Get Started" বাটনে ক্লিক করুন
4. এখন আপনি রুম add, edit, delete করতে পারবেন!

## সমস্যা সমাধান

### ব্যাকএন্ড স্টার্ট হচ্ছে না

- MongoDB connection string সঠিক আছে কিনা চেক করুন
- পাসওয়ার্ডে বিশেষ চিহ্ন থাকলে encode করুন (% দিয়ে)
- Network Access এ "Allow Access from Anywhere" সিলেক্ট করা আছে কিনা দেখুন

### ফ্রন্টএন্ড "Failed to load rooms" দেখাচ্ছে

- ব্যাকএন্ড সার্ভার রান হচ্ছে কিনা চেক করুন (port 5000)
- `http://localhost:5000/api/health` খুলে দেখুন সার্ভার রেসপন্স দিচ্ছে কিনা

### "Port already in use" এরর

- অন্য কোনো অ্যাপ্লিকেশন একই পোর্ট ব্যবহার করছে
- ব্যাকএন্ডের `.env` ফাইলে PORT পরিবর্তন করুন (যেমন: 5001)
- ফ্রন্টএন্ডের `.env.local` এ API_URL আপডেট করুন
